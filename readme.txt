Nepali Forex Rate plugin displays 5 forex rate (USD, EUR, AUD, CHF, GBP) in tabular format based on Google Finance. 

== Description ==

This plugin displays 5 forex rate (USD, EUR, AUD, CHF, GBP) in tabular format based on Google Finance. It is very simple tool.You can just use [np_forex_rate] short code where you want to display the table. It shows exchange rate of foreign currencies for Nepali Rupees in real time so it is suitable for your blog or website.


== Installation ==

1. Download, unzip and upload to your WordPress plugins directory
2. activate the plugin within you WordPress Administration
